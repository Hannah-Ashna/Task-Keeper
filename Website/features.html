Main Features

As students, we can resonate when procrastination takes over. So to combat this, we have collectively produced an application that cancels out the need for this (or so we hope!). The aim of this app is to help students be more productive when it come to academic work. We aim to achieve this by creating an application that allows the user to earn in-game points to look after a virtual pet. When using our app, you can expect the following features:

  • Easy to use GUI.
  • Create, edit and delete a task list.
  • A points system based on the type of task and the speed in which it was completed.
  • These points can then be used to make in-game purchases for their pet.
  • A customisable pet.
  • Modify pets details.
  • The pets stats will be decreased when a deadline is not met.
  • Create multiple task lists and categorise them.
  • Set importance level for different tasks meaning more points for more important tasks completed on time.
  • User should be able to share their pet via social media.
